Add-ons

Krang's add-on system allows new features to be installed into Krang. The following add-ons are available for download:
  • Krang-BricLoader - load data from Bricolage into Krang
  • Krang-RSS - Element addon that creates RSS files to help syndicate your website.
  • Krang-SiteMap - automated creation of robot-friendly sitemaps for Search Engine Optimization in Krang.
