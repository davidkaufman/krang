CVS

The first step in joining the Krang project is to checkout a copy of Krang from CVS. This is the best version to use to make patches you can send to the krang-devel mailing list.

Anonymous CVS

Anonymous CVS lets you checkout a copy of the source, but won't allow checkins. First, login: 
cvs -d:pserver:anonymous@krang-cvs.thepirtgroup.com:/usr/local/krang-cvs login
When you see a password prompt, just hit enter. Then you can checkout a copy of Krang: 
cvs -d:pserver:anonymous@krang-cvs.thepirtgroup.com:/usr/local/krang-cvs co -P krang

Developer CVS

If you're a core developer on the Krang project you'll be given a krang-cvs account. To use it, access CVS via SSH: 
export CVS_RSH=ssh
cvs -d:ext:USER@krang-cvs.thepirtgroup.com:/usr/local/krang-cvs co krang
